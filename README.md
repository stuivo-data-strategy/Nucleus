# Nucleus HR

## DEMO MOMENT — "This is not SAP"

1. Click the FAB ("+" button)
   → Decision tree appears with 4 claim type cards
   → "In SAP, you'd already be filling in a form. Here, we ask what you're doing first."

2. Select "Mileage Claim"
   → Point out the saved vehicles and frequent journeys
   → "Sarah's car is already known to the system. She doesn't re-enter it every time."

3. Enter a journey (or select a saved one)
   → Click "Calculate distance"
   → "The system calculates the distance — SAP trusts the user to enter it correctly."

4. Point out the mileage threshold callout
   → "Sarah has 1,250 miles left at the 45p rate. The system tracks this across her tax year."

5. Show the approval route preview updating as the amount changes
   → "The routing is live. It knows the org structure."

6. Dismiss without submitting — go back to type selector

7. Select "Group Expense"
   → Add 3 attendees using the people picker
   → "One receipt, multiple people. The system records who was there."
   → Show the per-head amount calculating automatically
   → Show the policy check: £75/head

8. Dismiss — return to dashboard
